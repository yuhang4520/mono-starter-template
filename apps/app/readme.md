# @your-project/app

uni-app multi-platform client application.

## Features

- Vue 3 + TypeScript
- Pinia state management
- tRPC client integration
- UnoCSS for styling
- Multi-platform support (WeChat Mini Program, H5, etc.)

## Development

```bash
# WeChat Mini Program
pnpm --filter @your-project/app dev:mp-weixin

# H5
pnpm --filter @your-project/app dev:h5

# Other platforms
pnpm --filter @your-project/app dev:mp-alipay
pnpm --filter @your-project/app dev:mp-baidu
# ... more platforms
```

## Build

```bash
# Build for WeChat Mini Program
pnpm --filter @your-project/app build:mp-weixin

# Build for H5
pnpm --filter @your-project/app build:h5
```

## Environment Variables

Copy `env/.env.example` to `env/.env.local` and configure:

- `VITE_SERVER_URL`: API server URL (e.g., http://localhost:4000)

## Type Check

```bash
pnpm --filter @your-project/app type-check
```

## Adding Pages

Pages are auto-generated by `vite-plugin-uni-pages`. Add new `.vue` files to `src/pages/` directory.
